## 📂 강의 내용 개요

### 1. 알고리즘 개념
- **알고리즘(Algorithm)**: 문제를 해결하기 위한 명확하고 단계적인 절차

### 2. 알고리즘 학습 주제
- 문제 이해  
- 알고리즘 이해  
- 효율성 분석 (시간, 메모리 기준)  
- 추가 학습: 구현 및 문제 해결:contentReference[oaicite:1]{index=1}

### 3. 알고리즘 분석

### 4. 설계 기법
- Divide and Conquer (분할 정복)  
- Dynamic Programming (동적 프로그래밍)  
- Greedy (탐욕 기법)  

### 5. 주요 알고리즘
- Sorting (정렬)  
- Searching (탐색)  
- NP 문제 개요
